How to Use

All fields are joined by AND operation. Use % to match and filter name and roll number. Leaving everything empty will give result of all students.

Eg: Find all Student is CSE (B Tech) department of 2017 batch who has Rohit in their name.

Name as matched by substring. Roll No of CSE 2017 batch starts with 175. We will use % operator to match anything after this prefix.

So we can write
Student Name : Rohit
Roll No : 175%
